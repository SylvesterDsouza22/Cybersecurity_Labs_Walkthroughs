# 🧪 Proving Grounds Labs (OffSec)

## 📌 Overview

This section contains my hands-on walkthroughs from **Offensive Security Proving Grounds**, where I practice real-world penetration testing techniques on vulnerable machines.

Each lab is approached with a structured methodology similar to real penetration testing engagements.

---

## 🎯 Objective

The objective of these labs is to:

* Develop practical penetration testing skills
* Understand real-world vulnerabilities and exploitation techniques
* Improve problem-solving and analytical thinking
* Prepare for Offensive Security certifications and real job roles

---

## 🧠 Approach & Methodology

Each machine is solved using a step-by-step methodology:

### 🔍 1. Enumeration

* Nmap scanning (service & version detection)
* Directory and file discovery
* Identifying attack surfaces

---

### 🌐 2. Initial Access

* Web application exploitation
* Credential discovery
* Exploiting misconfigurations

---

### ⚡ 3. Privilege Escalation

* Local enumeration
* Weak permissions & misconfigured services
* Exploiting system vulnerabilities

---

### 👑 4. Post Exploitation

* Gaining root/admin access
* Capturing flags
* Understanding system weaknesses
