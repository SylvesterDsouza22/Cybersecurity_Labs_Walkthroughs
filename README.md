# 💣 Offensive Security Labs (OffSec)

## 📌 Overview

This repository contains my hands-on practice and walkthroughs from **Offensive Security (OffSec) labs**, including machines from **Proving Grounds** and other penetration testing environments.

The focus of this repository is to demonstrate **real-world offensive security skills**, including:

* Enumeration
* Exploitation
* Privilege Escalation
* Post-Exploitation Techniques

---

## 🎯 Objective

The goal of these labs is to develop a strong foundation in **penetration testing** by solving vulnerable machines in a structured and methodical way.

Each walkthrough follows a practical approach similar to real-world engagements.

---

## 🧠 Methodology

Each machine is approached using the following methodology:

1. 🔍 **Enumeration**

   * Nmap scanning
   * Service identification
   * Directory brute-forcing

2. 🌐 **Initial Foothold**

   * Web exploitation
   * Credential discovery
   * Misconfiguration abuse

3. 💻 **Privilege Escalation**

   * Local enumeration
   * Misconfigured files/services
   * Exploiting weak permissions

4. 👑 **Post-Exploitation**

   * Root/user access verification
   * Flag capture
   * System analysis
